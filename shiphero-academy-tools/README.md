# ShipHero Academy Tools

Static prototype site for GitHub Pages.

## Publish with GitHub Pages

1. Upload this folder's contents to the root of your repository.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select **main** and **/(root)**, then save.

The landing page links to both tools. Each tool keeps its original JavaScript functionality and receives shared Academy styling from `assets/academy.css`.
